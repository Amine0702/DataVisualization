# Data Visualization with Matplotlib

## Project Description
This project demonstrates the use of **Matplotlib** in Python for visualizing data.
It covers a variety of plots including line plots, multiple line plots, bar charts, stacked bar charts, and customizations.

## Objectives
- Visualize mathematical functions such as sine and cosine
- Display categorical and numerical data using bar charts
- Apply plot customizations: colors, markers, linewidth, legends, titles, axis labels
- Understand stacked bar charts with error bars


## Libraries Used
- Python
- numpy
- matplotlib
- pandas

## Examples of Plots
1. **Line Plots**
   - Simple line plots
   - Multiple lines on one plot
   - Custom markers, colors, and styles
2. **Bar Charts**
   - Simple bar charts
   - Grouped bar charts
   - Stacked bar charts with error bars
3. **Customizations**
   - Titles, axis labels, legends
   - Axis limits
   - Colors, markers, linewidth
